# ポートフォリオ

日本の IT 専門学校に在学中の留学生です。
フロントエンドとバックエンドの両方を学びながら、実際に動作する Web アプリケーションを独力で開発しています。
現在は Java / Spring Boot を中心としたバックエンド開発 と Vue 3 を用いたフロントエンド開発 に力を入れています。  

## 主力プロジェクト：駐車場管理システム（フルスタック独立開発）

簡単な車輌使用、乗車、降車管理などのシステムです。  
フロントエンドとバックエンドを完全に分離し、実際の開発現場に近い構成で実装しました。

### 技術スタック
**フロントエンド**
- Vue 3（Composition API）
- TypeScript
- Vite
- Pinia（状態管理）
- Element Plus（UI コンポーネント）
- Axios（API 通信）

**バックエンド**
- Spring Boot
- Spring Security 
- MyBatis
- MySQL
- JWT 認証
- Lombok

**その他**
- 前後端分離アーキテクチャ
- RESTful API 設計
- 統一レスポンス形式
- ページネーション・条件検索
- 例外処理
